## 🎯 Project Overview

**NexusStore** is a standalone e-commerce website designed for modern retail. This project delivers premium UI/UX, real-world shopping functionality, and bold visual design using only frontend technologies.

**Project Type:** BCA Final Year Project / Portfolio Showcase  
**Development Time:** 8-10 hours  
**Hosting:** 100% Free via GitHub Pages

---

## ✨ Strong Features & Special Options

### **1. Premium Shopping Experience**
- 🛒 **Smart Cart System** - Add/remove items, quantity update, persistent localStorage
- ❤️ **Wishlist** - Save products for later with one-click heart icon
- ⚡ **Quick View Modal** - Preview full product details + stock bar + sold count
- 🎟️ **Advanced Promo Codes** - `NEXUS50`, `MEGA100`, `FIRST20`, `FLASH30` with min cart rules
- 📦 **Order Management** - Complete order history + PDF invoice export via jsPDF
- 🔍 **Live Search + Voice Search** - Type or speak to search 10,000+ products
- 🏷️ **Category Filters** - Electronics, Fashion, Books, Home with filter pills

### **2. Bold UI/UX Design**
- **Animated Gradient Background** - Dynamic 15s color shift for premium feel
- **Glassmorphism Header** - Frosted glass navigation that blurs on scroll
- **Hero Banner** - Full-width gradient with animated CTA and live stats
- **Deal of the Day** - Live countdown timer + massive discount badges
- **Product Cards** - Hover-lift + glow + quick action buttons on hover
- **Toast Notifications** - Animated popups for all cart/wishlist actions
- **Dark Mode** - One-click toggle with saved preference
- **Skeleton Loaders** - Shimmer effect for better perceived performance
- **Confetti Animation** - Celebrates successful purchase

### **3. Advanced E-commerce Logic**
- ⭐ **Star Ratings** - Visual 5-star system + view count tracking
- 📊 **Smart Sorting** - Price low-high, high-low, top rated
- 👤 **Profile Dropdown** - Orders, admin panel, export PDF in one menu
- 🔐 **Admin Panel** - Add products live with name, price, MRP, stock, image, description
- 📱 **Fully Responsive** - Optimized for mobile, tablet, and desktop
- 💳 **Secure Checkout Flow** - Professional multi-step checkout with confetti
- 👁️ **Recently Viewed** - Sticky bar showing last 5 viewed products
- 📈 **Stock Management** - Progress bars + "Only X left" urgency + sold counter

### **4. Power User Options**
- **Voice Search** - Click mic icon and speak product name
- **Product Views Tracking** - Counts how many times each product was viewed
- **Sold Counter** - Shows real sales numbers on each product
- **Stock Progress Bar** - Visual indicator of inventory levels
- **Quick Actions** - Hover product to see view + add to cart buttons
- **Chat Widget** - Floating support button with pulse animation

## 🛠️ Tech Stack

| Technology | Purpose |
| --- | --- |
| **HTML5** | Semantic structure, modals, forms, accessibility |
| **CSS3** | Glassmorphism, gradients, keyframe animations, Grid/Flexbox |
| **Vanilla JavaScript** | Cart logic, auth, search, filters, localStorage, Web Speech API |
| **jsPDF** | Export order invoices as PDF files |
| **Google Fonts** | Inter + Outfit for bold, modern typography |
| **Material Icons** | Consistent icon system across UI |

**Zero Dependencies. No npm install. No build step. Just open `index.html`.**
